Transcriber Project - Whisper 音声文字起こしシステム
=====================================================

【概要】
--------
このツールは OpenAI Whisper を使用して、音声ファイル（.mp3）を文字起こしし、
箇条書き＋空白行の形式でテキスト出力するPythonベースのシステムです。

【起動手順】
-------------
1. 仮想環境の有効化（Windowsの場合）:
   > .\venv\Scripts\Activate

2. Pythonスクリプトの実行（例: transcribe.py がある場合）:
   > python transcribe.py

【使用方法】
--------------
1. 音声ファイルの設置:
   - 音声ファイル（.mp3）は以下のディレクトリに置いてください:
     `./audios/`
   - ファイル名は日本語でも問題ありませんが、半角英数字推奨です。

2. 出力ファイルの保存先:
   - 出力された文字起こしテキストは以下に保存されます:
     `./outputs/`
   - 形式は `.txt` で、元のファイル名に基づいた命名になります。
   - 各行は「・○○○○」の箇条書き形式＋行間に空白行が挿入されます。

3. Whisper モデルについて:
   - Whisper は初回起動時にモデルを自動ダウンロードします。
   - デフォルトでは "base" モデルが使用されます。必要に応じてコードを編集して "small" や "medium" モデルにも変更可能です。

【注意事項】
--------------
- 対応フォーマット: mp3, wav, m4a など Whisper が対応している形式。
- 出力精度は音声の明瞭さや話者の滑舌、BGMの有無により変動します。
- ファイルサイズが大きい場合、処理に時間がかかります（数分〜十数分）。

【トラブルシューティング】
-----------------------------
- 依存関係に問題がある場合は以下を再実行:
   > pip install git+https://github.com/openai/whisper.git

- GPUを使用したい場合は PyTorch の CUDA 対応バージョンを導入してください。
